# JavaScript Exercises

**This repository contains all the projects that are part of the JavaScript exercises from the Factoría F5 bootcamp**

## Project Structure

**Each exercise has its own folder inside this repository**
- Inside each folder you will find:

  - `src/` → source code (HTML, CSS/SASS, JS)
  - `tests/` → automated tests
  - `README.md` → explanation of the exercise
